# 小呆 IP

## 角色定义

小呆是中文正文配图的固定视觉 IP。
默认每张图都要出现小呆。小呆不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的荒诞工作者。

## 外形特征（必须保留）

- **发型**：黑色蓬松短发，微乱，发梢向右扫，露出额头
- **眼镜**：细框圆形金属眼镜，透明镜片
- **表情**：宁静、专注、温和，从不悲伤或夸张
- **服装**：黑色宽松 T 恤，圆领无包边，左胸有白色小火苗图标，宽松长裤
- **体型**：Q 版比例，大头小身，圆润但保持辨识度
- **姿态**：盘腿而坐，双手自然交叠于腿上
- **线稿风格**：黑色简笔手绘，轻微抖动，不写实，不矢量

## 绝对禁止

- 火苗、能量球、火焰等任何发光道具（除胸前小火苗图标外）
- 金色卵石、水晶、岩石等周围装饰
- 领子、纽扣、口袋等服装细节，T恤上禁止任何额外图案
- 写实渲染、复杂阴影、渐变
- 可爱卡通、表情包、儿童插画
- 悲伤、戏剧化、夸张的表情或动作
- 画面出现任何手写文字、标注、字母、符号
- 画面出现任何非角色的额外物件（除了角色的衣服和身体，没有任何东西）

## 适用场景

作为正文配图的核心角色，承担画面中的概念动作（如：观察断点、拉线、分拣、承接等）。

## 判断标准

如果去掉小呆，图的核心隐喻还能完全成立，说明小呆太装饰了；要重写提示词，让小呆成为动作主体。
